# 🌐 Phase 1 – Part 2: Networking Fundamentals (CCNA Foundation for VAPT)

## 📖 Overview

This section builds the networking foundation required for Vulnerability Assessment and Penetration Testing (VAPT). A strong understanding of networking is essential for reconnaissance, scanning, enumeration, web application testing, and network security assessments.

This part focuses on core networking concepts, practical labs, packet analysis, and basic network troubleshooting.

---

# 🎯 Learning Objectives

After completing this part, I will be able to:

* Understand networking fundamentals.
* Explain the OSI and TCP/IP models.
* Understand Ethernet communication.
* Configure and identify IPv4 and IPv6 addresses.
* Understand subnetting and CIDR notation.
* Differentiate between private and public IP addresses.
* Explain MAC addresses and ARP.
* Understand DNS, DHCP, and NAT.
* Differentiate TCP and UDP.
* Identify common ports and protocols.
* Understand switching and routing basics.
* Understand VLAN fundamentals.
* Capture and analyze packets using Wireshark.
* Build basic network topologies using Cisco Packet Tracer.
