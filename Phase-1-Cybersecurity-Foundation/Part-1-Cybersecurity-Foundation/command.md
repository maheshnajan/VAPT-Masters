**🪟 Windows Basic Commands**

| Command                  | Purpose                 |
| ------------------------ | ----------------------- |
| `hostname`               | Computer Name           |
| `whoami`                 | Current Logged-in User  |
| `systeminfo`             | System Information      |
| `ipconfig`               | IP Configuration        |
| `ipconfig /all`          | Detailed IP Information |
| `ipconfig /release`      | Release DHCP IP         |
| `ipconfig /renew`        | Renew DHCP IP           |
| `ipconfig /flushdns`     | Clear DNS Cache         |
| `ipconfig /displaydns`   | Show DNS Cache          |
| `ping`                   | Check Connectivity      |
| `tracert`                | Trace Packet Route      |
| `pathping`               | Packet Loss Analysis    |
| `arp -a`                 | View ARP Cache          |
| `netstat -ano`           | Active Connections      |
| `nslookup`               | DNS Lookup              |
| `route print`            | Routing Table           |
| `tasklist`               | Running Processes       |
| `taskkill /PID <PID> /F` | Kill Process            |
| `net user`               | Local Users             |
| `net localgroup`         | Local Groups            |
| `net share`              | Shared Folders          |
| `net use`                | Network Shares          |
| `wmic os get caption`    | Windows Version         |
| `echo %USERNAME%`        | Username                |
| `echo %COMPUTERNAME%`    | Computer Name           |
| `dir`                    | List Files              |
| `cd`                     | Change Directory        |
| `cls`                    | Clear Screen            |
| `mkdir`                  | Create Directory        |
| `rmdir`                  | Remove Directory        |
| `copy`                   | Copy Files              |
| `move`                   | Move Files              |
| `del`                    | Delete Files            |
| `type`                   | Display File Content    |
| `findstr`                | Search Text             |
| `tree`                   | Folder Tree             |
| `shutdown /r /t 0`       | Restart System          |
| `shutdown /s /t 0`       | Shutdown System         |
