# 📘 Phase 1 – Part 1: Cybersecurity Foundation & VAPT Introduction

## 📖 Overview

This section builds the fundamental knowledge required before learning Networking, Linux, Web Security, and Penetration Testing.

The goal of this part is to understand the core concepts of Cybersecurity and the complete Vulnerability Assessment and Penetration Testing (VAPT) process.

---

# 🎯 Learning Objectives

After completing this part, I will be able to:

* Understand the fundamentals of Cybersecurity
* Explain Information Security concepts
* Differentiate between Threat, Vulnerability, and Risk
* Explain the CIA Triad
* Understand Security Controls
* Identify different types of Hackers
* Understand the VAPT lifecycle
* Differentiate Vulnerability Assessment and Penetration Testing
* Understand different Pentesting methodologies
* Build a strong foundation for upcoming VAPT topics
