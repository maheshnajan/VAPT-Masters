# 📘 Phase 1 – Part 1: Cybersecurity Foundation & VAPT Introduction

## 📖 Overview

This section builds the fundamental knowledge required before learning Networking, Linux, Web Security, and Penetration Testing.

The goal of this part is to understand the core concepts of Cybersecurity and the complete Vulnerability Assessment and Penetration Testing (VAPT) process.
